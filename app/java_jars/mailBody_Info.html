CHEMDNER news!
Dear BioCreative participant,

We send you some important update on the CHEMDNER-patents task.

  1. The development gold standard annotations for the CHEMDNER tasks are now available at Markyt Prediction Evaluation tool (http://markyt.org/biocreative/analysis).
  2. We have prepared a baseline prediction for the development sets of the CEMP and GPRO tasks using a list lookup approach (case sensitive term lookup of the names in the training applied to the development set).
  3. Important: In order to make sure that you can retrieve the test set release and to be able to upload your test set predictions please be so kind to test your access credentials of the Markyt system as soon as possible.

    Bear in mind that teams' final predictions for the competition will be submitted in Markyt. You will need to upload your predictions by additionally specifying your team-id, task name (cemp, cpd, gpro) and run name (up to five runs per task, i.e. 1-5).

  4. The test set patent records will be released the: August 14th 2015. It will consist of 40,000 records that contain the subset of 7000 records that will be used as test set. The other records have been added as a background set to make sure that the systems are totally automatic and that no manual corrections of the test set predictions can be done.
  5. Additional resources: we have compiled a list of additional resources that might be useful for participating teams at: http://www.biocreative.org/tasks/biocreative-v/chemdner-patents-task-resource/
  6. Submission format: please make sure that your submission is in the right format, that you provide the requested data in each of the prediction columns and that you do not have any duplicate predictions.

Please be so kind to test your access credetials as soon as possible. Bear in mind that teams' final predictions for the competition will be submitted in Markyt.

Please do not reply to this mail.

Kind regards,
